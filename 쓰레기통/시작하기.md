## 🎨 CSS 파일 가이드

### 어떤 파일을 수정해야 할까?

| 수정하고 싶은 것 | 파일 |
|----------------|------|
| 색상 변경 | `css/base.css` |
| 레이아웃 변경 | `css/layout.css` |
| 버튼 스타일 | `css/components.css` |
| 프로필 페이지 | `css/profile.css` |
| 작품 그리드 | `css/artwork.css` |
| 업로드 모달 | `css/upload.css` |

### 예시: 색상 테마 변경

```css
/* css/base.css */
:root {
    --primary: #8b5cf6;      /* 파란색 → 보라색 */
    --primary-hover: #7c3aed;
}
```

→ 전체 앱의 파란색이 보라색으로 변경됨!

## 🚀 Cursor 사용 방법

### 이제 이렇게 명령하세요!

#### ✅ 좋은 명령 예시

```
1. "프로필 아바타 크기를 200px로 키워줘"
   → Cursor: css/profile.css 수정

2. "작품 그리드를 4열로 변경해줘"
   → Cursor: css/artwork.css 수정

3. "앱 전체 색상을 보라색으로 바꿔줘"
   → Cursor: css/base.css 수정

4. "업로드 모달 디자인 개선해줘"
   → Cursor: css/upload.css 수정

5. "모바일에서 레이아웃 수정해줘"
   → Cursor: css/layout.css 미디어 쿼리 수정
```

#### 📁 파일 참조 사용

```
"@css/profile.css 에서 아바타 크기 변경해줘"
"@css/artwork.css 에서 그리드 간격 늘려줘"
"@PROJECT_STRUCTURE.md 를 보고 스타일 수정해줘"
```
